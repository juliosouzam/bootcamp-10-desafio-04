## :memo: Desafio

O desafio é criar uma página semelhante ao feed do facebook, com posts e comentários.

## :computer: Instalação

```sh
# clonar o repositório
$ git clone https://github.com/juliosouzam/bootcamp-10-desafio-04.git
$ cd bootcamp-10-desafio-04

# instalar os pacotes
$ yarn install

# iniciar a aplicação
$ yarn start
```
